# 📱 DSA Phonebook

A Java-based phonebook application built using core **Data Structures and Algorithms** concepts, simulating the contact management system of a mobile phone.

---

## 📋 About The Project

This project was developed as part of a **Data Structures and Algorithms** course at NUST. The goal was to apply fundamental DSA concepts to build a functional phonebook that can store, search, update, and delete contacts — the same way a real mobile phone manages its contacts.

---

## ✨ Features

- ➕ Add new contacts (name and phone number)
- 🔍 Search for a contact by name
- 📋 Display all contacts in sorted order
- ✏️ Update an existing contact's details
- 🗑️ Delete a contact
- 📊 Efficient storage using appropriate data structures
